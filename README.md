# 运行条件
- 运行 `pip install -r requirements.txt`
- Linux 用户需安装 imagemagick: `sudo apt-get install imagemagick`

# 运行代码
[lsp.py](src/lsp.py) 为主程序，直接运行即可: `python lsp.py`
